# RAG 기반 추천 시스템 개발 계획

## 프로젝트 개요
- LLM(Gemini)을 활용한 범용 추천 엔진
- 자연어 질의 → 시맨틱 검색 → LLM 답변 생성
- VS Code 터미널에서 실행 가능한 데모 포함

## 기술 스택
- 언어: Python
- LLM: Gemini
- 벡터DB: ChromaDB
- 임베딩: Gemini (text-embedding-004)
